Global mode

This defines the main criteria for removing Origins. Both modes should be perfectly safe, but the aggressive mode is less tolerant, therefore it causes more web content to break. The relaxed mode should break website functionality only very rarely.


Overrides

2 = aggressive
1 = relaxed
0 = whitelisted


Strict resource types

Origin headers will always be removed from GET requests of the types selected here. These rules are only meaningful in relaxed mode.
